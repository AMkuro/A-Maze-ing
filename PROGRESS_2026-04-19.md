## Mandatory Part

### IV.2 Usage

- [x] `python3 a_maze_ing.py config.txt` で起動できる
- [x] 引数が不正な場合はエラーメッセージを表示して終了する
- [x] 予期しないクラッシュが発生しない

### IV.3 Configuration file format

- [x] `KEY=VALUE` 形式を1行1ペアで読み込む
- [x] `#` コメント行を無視する
- [x] 必須キー `WIDTH` / `HEIGHT` / `ENTRY` / `EXIT` / `OUTPUT_FILE` / `PERFECT` をすべて処理する
- [x] 任意キー `SEED` / `ALGORITHM` / `DISPLAY_MODE` を受け付ける
- [x] デフォルト設定ファイル `config.txt` がリポジトリに存在する
- [x] 不正なキー・重複キー・範囲外値はエラーメッセージを表示して終了する

### IV.4 Maze Requirements

- [x] ランダム生成される
- [x] `SEED` 指定で再現性がある
- [x] 各セルが N / E / S / W の壁情報を持つ
- [x] `ENTRY` と `EXIT` が異なり、迷路内に存在する
- [ ] **全連結性チェックがない**（`MazeValidator` に connectivity 検証が未実装）
- [x] 隣接セル間の壁整合性を検証する（`_check_wall_consistency`）
- [x] 外周はすべて壁で閉じている（`_check_outer_walls`）
- [x] 3×3 の完全開放エリアを作らない（生成時チェック + `_check_no_open_3x3`）
- [x] "42" パターンを迷路中央に埋め込む
- [ ] **"42" 不可能時のコンソールエラーメッセージがない**（`_embed_42_pattern` が `print` せずに `return`）
- [x] `PERFECT=True` のとき完全迷路を生成する（DFS が spanning tree を生成する）
- [ ] **`PERFECT=True` の検証が未実装**（`MazeValidator` に perfect チェックがない）

### IV.5 Output File Format

- [x] 1セル1桁の16進数で出力する
- [x] 行ごとに1行、末尾 `\n`
- [x] 空行の後に入口座標・出口座標・最短経路 NEWS を出力する
- [x] 最短経路は BFS による正しい最短経路

## Common Instructions (Chapter III)

### III.1 General Rules

- [x] Python 3.10 以上
- [ ] **flake8 準拠の確認が必要**（`make lint` で要確認）
- [x] 例外は try-except で捕捉してクラッシュしない
- [x] ファイルハンドルはコンテキストマネージャ（`with open`）で管理している
- [x] 型ヒントを全関数に付与している
- [ ] **mypy エラーがないか要確認**（`make lint` で要確認）
- [x] docstring を記述している

### III.2 Makefile

- [x] `make install` — 仮想環境作成 + 依存インストール
- [x] `make run` — メインスクリプト実行
- [x] `make debug` — pdb でデバッグ実行

- [x] `make lint` — flake8 + mypy（必須フラグ）
- [x] `make lint-strict` — flake8 + mypy --strict
